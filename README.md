# [`curtisfree.com`][site]

## Building with Jekyll

### Setup

On macOS, with [Homebrew][brew] installed:

```sh
brew install ruby
gem install bundler
bundle install
```

### Generate, Serve, & Test

```sh
bundle exec jekyll serve --watch
```

Generated static content will be in `_site/` and hosted at `http://localhost:4000`.
